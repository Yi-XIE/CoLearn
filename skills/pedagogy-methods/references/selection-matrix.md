# Selection Matrix

## By Learning Goal

Foundational knowledge:
- Primary: explicit instruction, worked examples, retrieval practice.
- Support: formative assessment, spaced practice, UDL.

Procedural fluency:
- Primary: modeling, guided practice, deliberate practice.
- Support: feedback loops, interleaving after basics are stable.

Conceptual understanding:
- Primary: inquiry, concept comparison, examples and non-examples.
- Support: formative assessment, explanation prompts, metacognitive checks.

Transfer and authentic performance:
- Primary: project-based learning, problem-based learning, case-based learning.
- Support: scaffolding, rubrics, critique and revision.

Creativity and design:
- Primary: project-based learning, studio critique, design thinking.
- Support: constraints, prototypes, peer feedback, reflection.

Collaboration and communication:
- Primary: collaborative learning, discussion protocols, peer instruction.
- Support: roles, individual accountability, formative checks.

Self-directed learning:
- Primary: metacognition and self-regulated learning.
- Support: goal setting, planning prompts, progress dashboards, spaced review.

Inclusion and accessibility:
- Primary: UDL, differentiated instruction.
- Support: multimodal resources, adjustable challenge, assistive options.

## By Learner Readiness

Novice learners:
- Use more explicit instruction, worked examples, guided practice, and frequent checks.
- Delay open-ended inquiry until core schema exists.

Mixed-readiness groups:
- Use UDL, differentiation, flexible grouping, tiered tasks, and formative assessment.
- Keep the core target stable while varying support and product options.

Advanced learners:
- Use problem-based learning, project-based learning, inquiry extension, peer teaching, and independent investigation.
- Keep challenge authentic, not just more work.

## By Time Constraint

10-20 minutes:
- Retrieval practice, worked example, mini formative check, peer instruction.

30-60 minutes:
- Explicit instruction cycle, 5E micro-inquiry, guided practice, short collaborative task.

Multi-day:
- Inquiry sequence, problem-based learning case, writing workshop, design sprint.

Multi-week:
- Project-based learning, interdisciplinary unit, portfolio, community product.

## Combination Recipes

Teach a hard new concept:
- Explicit instruction -> worked examples -> guided practice -> formative check -> retrieval review.

Make knowledge stick:
- Retrieval practice -> feedback -> spaced review -> interleaving -> cumulative assessment.

Prepare for authentic application:
- Explicit mini-lessons -> scaffolded practice -> problem-based case -> project artifact -> reflection.

Design an inclusive digital learning flow:
- UDL options -> diagnostic check -> adaptive scaffolds -> formative feedback -> learner reflection.

Turn a lesson into an AI tutor flow:
- Goal diagnosis -> model one example -> ask learner to attempt -> detect misconception -> give targeted hint -> ask for retrieval later.
