# Method Cards

## 显性教学 Explicit Instruction

Use when learners are novices, content is procedural, misconceptions are common, or safety/accuracy matters.

Core pattern:
1. Activate prior knowledge with a brief review.
2. State the learning target and success criteria.
3. Model the process with think-aloud.
4. Use worked examples and non-examples.
5. Move from guided practice to independent practice.
6. Give immediate feedback and cumulative review.

Good for: mathematics, grammar, coding basics, laboratory procedure, foundational concepts.
Watch for: overlong teacher talk. Keep checks for understanding frequent.

## 脚手架 Scaffolding and Gradual Release

Use when learners can perform a task with support but not yet independently.

Core pattern:
1. Break the target performance into visible subskills.
2. Provide prompts, templates, sentence stems, hints, or partial worked examples.
3. Fade support as learner accuracy and confidence improve.
4. Transfer responsibility through "I do, we do, you do" or similar structures.

Good for: writing, problem solving, scientific explanation, project milestones.
Watch for: support that never fades. Add clear exit criteria.

## 项目式学习 Project-Based Learning

Use when the goal includes transfer, authentic problem solving, collaboration, communication, or public products.

Core pattern:
1. Start with a meaningful driving question or challenge.
2. Define academic goals and product criteria.
3. Run sustained inquiry with checkpoints.
4. Include critique, revision, reflection, and public sharing.
5. Assess both content learning and process quality.

Good for: interdisciplinary units, design challenges, civic issues, STEM projects.
Watch for: beautiful products with thin learning. Keep content instruction and assessment explicit.

## 问题式学习 Problem-Based Learning

Use when learners should diagnose an ill-structured problem and learn concepts through solving it.

Core pattern:
1. Present a realistic problem before full instruction.
2. Have learners identify what they know, need to know, and possible hypotheses.
3. Support inquiry, resource use, and expert-like reasoning.
4. Debrief the solution path and formalize the underlying concepts.

Good for: medicine, engineering, business cases, ethics, policy, applied science.
Watch for: novices floundering. Add guiding questions and targeted mini-lessons.

## 探究式学习 Inquiry-Based Learning

Use when the goal is scientific reasoning, curiosity, evidence use, and conceptual change.

Core pattern:
1. Engage with a phenomenon or puzzling question.
2. Explore through observation, experiment, or data.
3. Explain with claims, evidence, and reasoning.
4. Elaborate by applying the idea to a new case.
5. Evaluate through performance or reflection.

Good for: science, geography, social science, data literacy.
Watch for: unguided discovery. Sequence exploration with explicit consolidation.

## UDL Universal Design for Learning

Use when designing for learner variability, accessibility, motivation, and inclusive participation.

Core pattern:
1. Offer multiple ways to access information.
2. Offer multiple ways to express understanding.
3. Offer multiple ways to engage and persist.
4. Remove unnecessary barriers while keeping the learning goal rigorous.

Good for: mixed-ability classrooms, multilingual learners, special education support, digital products.
Watch for: too many choices without structure. Offer bounded options.

## 差异化教学 Differentiated Instruction

Use when learners differ meaningfully in readiness, interest, language, pace, or support needs.

Core pattern:
1. Diagnose readiness and barriers.
2. Adjust content, process, product, or learning environment.
3. Keep the same essential learning target where possible.
4. Use flexible grouping and targeted support.

Good for: mixed-level classes, remediation, enrichment, adaptive learning products.
Watch for: tracking students into fixed levels. Reassess frequently.

## 检索练习 Retrieval Practice

Use when the goal is durable memory and fluent access to knowledge.

Core pattern:
1. Ask learners to recall without looking.
2. Use low-stakes quizzes, brain dumps, flashcards, or oral prompts.
3. Give feedback quickly.
4. Revisit across time and mix old with new material.

Good for: vocabulary, facts, formulas, definitions, procedures, key concepts.
Watch for: turning practice into high-stakes testing. Keep it frequent and low pressure.

## 间隔复习 Spaced Practice

Use when knowledge must last beyond a single lesson.

Core pattern:
1. Revisit material after increasing intervals.
2. Combine old and new items in short sessions.
3. Use learner performance to schedule review.

Good for: language learning, exam preparation, mathematics fluency, professional training.
Watch for: review that is too easy. Include effortful recall.

## 交错练习 Interleaving

Use when learners must choose the right strategy among similar problem types.

Core pattern:
1. Mix related problem types after initial instruction.
2. Ask learners to identify which strategy fits and why.
3. Compare examples and boundary cases.

Good for: math problem types, grammar forms, art styles, diagnostic reasoning.
Watch for: using it too early. Teach basics first, then mix.

## 形成性评价 Formative Assessment

Use when teaching needs rapid evidence to adjust instruction.

Core pattern:
1. Define the evidence needed for the learning target.
2. Gather it through exit tickets, mini-whiteboards, quick quizzes, concept maps, or short performances.
3. Interpret patterns, not only individual scores.
4. Reteach, regroup, extend, or move on.

Good for: every lesson flow and AI tutor loop.
Watch for: collecting data without changing instruction. Always specify the decision rule.

## 元认知与自我调节 Metacognition and Self-Regulated Learning

Use when learners need to plan, monitor, and evaluate their own learning strategies.

Core pattern:
1. Model expert thinking explicitly.
2. Teach planning, monitoring, and evaluation prompts.
3. Let learners select strategies and reflect on effectiveness.
4. Connect reflection to the next action.

Good for: independent study, writing, problem solving, long projects, AI tutor coaching.
Watch for: generic reflection. Tie prompts to a concrete task and evidence.

## 协作学习 Collaborative Learning

Use when peer interaction can deepen reasoning, explanation, critique, or social belonging.

Core pattern:
1. Give a task that truly needs collaboration.
2. Assign roles, norms, and individual accountability.
3. Structure talk with prompts or protocols.
4. Assess both shared product and individual understanding.

Good for: debate, design critique, peer instruction, project work, reading circles.
Watch for: unequal participation. Use visible roles and quick individual checks.

## 翻转课堂 Flipped Learning

Use when direct content exposure can happen before class and live time is better spent on practice, feedback, or application.

Core pattern:
1. Provide short pre-class input with accountability.
2. Start class with diagnostic checks.
3. Use class time for guided practice, collaboration, or problem solving.
4. Offer support for learners who did not complete preparation.

Good for: higher education, skill practice, discussion-heavy courses, blended learning.
Watch for: assuming all learners prepared. Build an in-class recovery path.
