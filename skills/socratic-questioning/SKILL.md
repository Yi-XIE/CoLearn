# Socratic Questioning

## When to Use

- Student gives a correct or incorrect answer without showing reasoning
- Student holds a misconception (critical_blocker present)
- Turn mode: CORRECTION (challenge assumptions) or VERIFY (deepen reasoning)
- Cognitive load: any — questions can be calibrated to load level

## How to Apply

1. **Start with a clarifying question** — make the student state their claim explicitly.
   - "你说 X 是对的——你是怎么得出这个结论的？"
   - "What makes you think that's the case?"

2. **Probe assumptions** — identify what the student takes for granted.
   - "如果 [assumption] 不成立呢？结论还对吗？"
   - "What would need to be true for your answer to hold?"

3. **Explore consequences** — follow the student's logic to its endpoint.
   - "如果按你的理解，那 [edge case] 会怎样？"
   - "If that's true, what would happen when [edge case]?"

4. **Present a counterexample** (if needed) — let the student reconcile.
   - "这里有一个例子似乎跟你的解释矛盾——你怎么看？"

5. **Let the student conclude** — don't give the answer; guide them to it.

## Question Types (Taxonomy)

| Type | Purpose | Example |
|------|---------|---------|
| Clarification | Make implicit explicit | "你说的 X 具体指什么？" |
| Assumption | Surface hidden premises | "这里你假设了什么？" |
| Evidence | Demand justification | "你怎么知道这是对的？" |
| Perspective | Consider alternatives | "有没有另一种解释？" |
| Consequence | Follow logic forward | "如果是这样，那会导致什么？" |
| Meta | Reflect on the question itself | "为什么这个问题重要？" |

## Example Dialogue

```
Student: 特征值就是矩阵的大小。
Tutor: 有意思——你说"大小"是什么意思？是矩阵的维度，还是某种度量？
Student: 就是...矩阵变换后向量变大了多少倍。
Tutor: 那如果特征值是负数呢？向量"变大了负倍"是什么意思？
Student: 嗯...可能方向反了？
Tutor: 对！所以特征值不只是"大小"——它同时编码了缩放和方向。
```
